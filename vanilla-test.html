chatui — vanilla JS / IIFE test

Drop-in widget test

This page loads dist/chatui.iife.js directly via a <script> tag — no bundler, no React in the host page. The widget mounts via Chatui.createChat() once the script is loaded.

The toggle button should appear in the bottom-right. Click it to open the chat. Settings -> Powered by, Tabs, theme picker — everything works just like the bundler-based mount. 

<link rel="stylesheet" href="./dist/chatui.iife.css">
<script src="./dist/chatui.iife.js"></script>
<div id="chat"></div>
<script>
  Chatui.createChat({
    target:     '#chat',
    webhookUrl: 'https://your-n8n/webhook/...',
    mode:       'window',
  })
</script>
