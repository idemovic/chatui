Integration Examples

This file demonstrates four ways to embed chatui into an existing project. Pick the one that fits your setup — they go from simplest to most flexible.

Option A — Custom button on your page (recommended)

You own the toggle button; the chat panel is an iframe that slides in/out.

Option B — Fullscreen chat page inside an iframe

Good for a dedicated support page or a portal tab.

Option C — Floating widget (iframe, window mode)

Set mode: "window" in chat-config.json. The iframe must be positioned where you want the button to appear.

Option D — Direct React mount (monorepo / same bundler)

// YourPage.tsx
import { App as ChatWidget } from 'chatui/src/App'
import { useSettingsStore }  from 'chatui/src/store/settingsStore'
import 'chatui/src/index.css'

// Configure once, before first render
useSettingsStore.getState().setConfig({
  webhookUrl: 'https://your-n8n.app.n8n.cloud/webhook/abc123',
  mode:        'window',     // or 'fullscreen' | 'mixed'
  botName:     'Support',
  theme:       'midnight',
  streaming:   true,
  showSidebar: false,
  showCta:     true,
  ctaDelay:    4000,
  i18n: {
    en: { ctaText: 'Hi! Need help?', initialMessages: ['Hello! How can I assist you today?'] },
  },
})

export function YourPage() {
  return (
    <div style={{ height: '100vh' }}>
      <YourOwnContent />
      <ChatWidget />  {/* floating button overlays the page */}
    </div>
  )
}
